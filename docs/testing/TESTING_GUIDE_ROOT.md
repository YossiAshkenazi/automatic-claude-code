## ü§ñ Automated Test Script

Create `test-v2.sh` (Linux/Mac) or `test-v2.ps1` (Windows):

### Windows PowerShell Script (test-v2.ps1)
```powershell
# Automatic Claude Code v2.0 Test Suite
Write-Host "üß™ Starting ACC v2.0 Test Suite" -ForegroundColor Cyan
Write-Host "================================" -ForegroundColor Cyan

# Test 1: Health Check
Write-Host "`nüìã Test 1: SDK Health Check" -ForegroundColor Yellow
& acc run --verify-claude-cli "test"
if ($LASTEXITCODE -ne 0) {
    Write-Host "‚ùå Health check failed" -ForegroundColor Red
    exit 1
}
Write-Host "‚úÖ Health check passed" -ForegroundColor Green

# Test 2: Basic Generation
Write-Host "`nüìã Test 2: Basic Code Generation" -ForegroundColor Yellow
"// Test file" | Out-File -FilePath test-gen.js
& acc run "add a sum function to test-gen.js" -i 1
if ((Get-Content test-gen.js | Select-String "function").Count -eq 0) {
    Write-Host "‚ùå Code generation failed" -ForegroundColor Red
    exit 1
}
Write-Host "‚úÖ Code generation passed" -ForegroundColor Green

# Test 3: Error Handling
Write-Host "`nüìã Test 3: Error Handling" -ForegroundColor Yellow
$env:ANTHROPIC_API_KEY = "invalid-test-key"
$output = & acc run "test" -i 1 2>&1 | Out-String
Remove-Item Env:\ANTHROPIC_API_KEY -ErrorAction SilentlyContinue
if ($output -notmatch "Invalid API Key") {
    Write-Host "‚ùå Error handling failed" -ForegroundColor Red
    exit 1
}
Write-Host "‚úÖ Error handling passed" -ForegroundColor Green

# Test 4: Session Management
Write-Host "`nüìã Test 4: Session Management" -ForegroundColor Yellow
$sessions = & acc history 2>&1 | Out-String
if ($sessions -notmatch "Session") {
    Write-Host "‚ö†Ô∏è No sessions found (expected on first run)" -ForegroundColor Yellow
} else {
    Write-Host "‚úÖ Session management working" -ForegroundColor Green
}

# Cleanup
Remove-Item test-gen.js -ErrorAction SilentlyContinue

Write-Host "`n‚úÖ All tests completed successfully!" -ForegroundColor Green
Write-Host "================================" -ForegroundColor Cyan
```

## üìä Expected Test Results

### Successful Test Indicators
- ‚úÖ SDK health check shows all green
- ‚úÖ Code generation creates/modifies files correctly
- ‚úÖ Error messages are clear and include recovery steps

### Common Issues and Solutions

| Issue | Solution |
|-------|----------|
| "acc command not found" | Run `npm link` in project directory |
| "Claude CLI not found" | Install: `npm install -g @anthropic-ai/claude-code` |
| "Invalid API Key" | Unset: `unset ANTHROPIC_API_KEY` or use valid key |
| "SDK not available" | Run `acc run --verify-claude-cli "test"` to diagnose |
| "Circuit breaker open" | Wait 30 seconds and retry |
